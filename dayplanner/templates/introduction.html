Introduction

Welcome to Matt's Task Planner! This is a django powered website created by Matt Skovranek.

About Task Planner

I created this website to provide myself with a custom planner. It is created for my own requirements, ie: the black and green 'AS400' aesthetic. This is a straight forward application whose purpose is to organize my daily tasks and projects' tasks. There are seven objects, some of which are linked, ie: Category-Task-SubTask-Counter, Question-Answer. The website acts upon these objects to simulate how one would actually organize their activity. The objects are as follows:


  • Category/Project

    For grouping tasks into either general categories such as Health or Writing, or into projects such as preparing for a Vacation.

  • Task

    Each Task has a start date and time, due date and time, priority number, and status as finished or unfinished. Set priority to '0' to omit it in the display. Due date and time cannot be before start chronologically. The user can track tasks by start day, deadline or category/project.

  • SubTask

    Each Task can have unlimited subtasks added to it, for dividing a large task into smaller tasks. Also has priority number and finished status.

  • Counter

    Each Subtask can have a tally added to it, for things such as keeping track of how many reps of forty pushups you've done today.

  • Question

    Benjamin Franklin would start the day by answering a daily question, always, "What good shall I do today?". Likewise, I strive to ascertain a daily resolution by answering some equally appealling question. Perhaps, what is a positive habit to start today?

  • Answer

    Each question for resolution needs to be answered. What good shall I do today? Call an old friend I haven't heard from in a while.

  • Motto

    Timon and Pumbaa said "hakuna matata". Bob McFerrin said, "Don't worry, be happy." Or for complicated situations, "Keep It Simple, Stupid."
